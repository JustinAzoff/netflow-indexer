About Search time¶

There are 4 different kinds of searches:

  1. A cold search returning zero results.
  2. A cold search returning many results.
  3. A warm search returning zero results.
  4. A warm search returning many results.

Warm searches that return zero results are much faster than cold searches that return many results. Repeated warm searches finish in a fraction of a second but that usage pattern is not a realistic benchmark. I’ve tried to make Search time above represent how long you would have to wait for typical searches to finish.
